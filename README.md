# Todo_App
 
I have created a Todo App with jwt token based authentication in  react native. 
Screenshots are available in the Screenshots folder.

=> Operations:

* Sign up 
* Log in 
* Create User profile
* View Users profile
* Edit User profile
* Delete User profile

=> Front_end:

* React-native
* Redux

=> Back-end:

* REST API
* Mongo DB
* Express
* Postman API
